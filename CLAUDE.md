# CLAUDE.md

## Overview

A JavaScript fundamentals assignment covering function declarations, parameters, return values, and DOM event listeners. Part of a web development course.

## Tech Stack

- Vanilla JavaScript (ES5), HTML5, CSS3
- No build tools, bundler, or dependencies
- Served directly in browser or via VS Code Live Server

## Project Structure

```
assignment.js   - all assignment code (5 tasks)
index.html      - task descriptions and script import
styles.css      - styling (purple/orange theme with CSS counters)
```

## How to Run

Open `index.html` in a browser. Tasks execute via `alert()` dialogs on page load; Task 3 triggers on click of its list item.

## Key References

- `assignment.js:4-10`  - Task 1: function definitions (with and without parameters)
- `assignment.js:13-14` - Task 2: direct function calls
- `assignment.js:17`    - Task 3: event listener on `#task-3`
- `assignment.js:20-22` - Task 4: string concatenation function
- `assignment.js:25`    - Task 5: call and alert result
- `index.html:37`       - script loaded at end of body (no defer/async needed)
